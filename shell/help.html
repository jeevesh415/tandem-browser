Wat is Tandem Browser?

Tandem Browser is een speciale browser waarin jij (Robin) en je AI assistent (Kees) samen browsen. Terwijl jij normaal browst, kan Kees meekijken, helpen, en zelfstandig acties uitvoeren — allemaal onzichtbaar voor websites.

🧀 Kees Panel

Je AI assistent heeft een eigen paneel waar hij kan chatten, activiteiten volgen, en screenshots bekijken.

🖍️ Tekenen & Annotatie

Teken direct op webpagina's om belangrijke elementen te markeren voor Kees.

🎙️ Voice Input

Praat tegen Kees via voice input — sneller dan typen!

🛡️ Stealth Browsing

Advanced anti-detectie zorgt ervoor dat websites Tandem niet van echte Chrome kunnen onderscheiden.

💾 Smart Memory

Tandem onthoudt alle websites die je bezoekt en kan patronen en veranderingen detecteren.

📸 Screenshots & PiP

Maak screenshots met annotaties en gebruik Picture-in-Picture mode voor altijd-zichtbaar overzicht.

💡 Pro Tip Druk op Cmd+? om de sneltoetsen overlay te openen — zo leer je Tandem supersnel!

Kees Panel — Je AI Assistent

Het Kees panel (Cmd+K) is waar je AI assistent leeft. Het heeft meerdere tabs:

Chat

Praat direct met Kees via tekst of voice. Hij kan je helpen met browsing, vragen beantwoorden, en acties uitvoeren.

Activity Log

Kees volgt al je browsing activiteit in real-time: welke pagina's je bezoekt, waar je op klikt, wat je intypt. Dit helpt hem context te begrijpen.

Screenshots

Alle screenshots die je maakt zijn hier zichtbaar met thumbnails. Klik om de volledige afbeelding te bekijken.

Memory

Kees onthoudt belangrijke informatie over websites die je bezoekt — perfect voor terugvinden van informatie.

💡 Voice Tip Houd Cmd+M ingedrukt om te praten. Laat los en Kees verwerkt je spraak automatisch!

Tekenen & Annotatie

Met Cmd+Shift+D schakel je teken mode in. Je kunt dan direct op webpagina's tekenen.

Beschikbare Tools

✏️ Vrije Lijn

Teken vrije vormen, cirkels, onderstrepen.

🔸 Rechthoeken

Markeer secties en blokken met rechthoeken.

⭕ Cirkels

Highlight specifieke elementen met cirkels.

📝 Tekst Labels

Voeg tekst notities toe aan je annotaties.

Kies uit verschillende kleuren: rood (standaard), geel, groen, en blauw. Druk op "📸 Snap voor Kees" om een screenshot met al je annotaties te maken.

Voice Input

Tandem heeft ingebouwde spraakherkenning voor Nederlandse (Nederlands-België) en Engelse input.

Hoe te gebruiken

  1. Druk op Cmd+M om voice input te activeren
  2. Je ziet een rode pulserende indicator — begin te praten
  3. Kees transcribeert je spraak live in het chat paneel
  4. Na een korte stilte wordt het bericht automatisch verzonden

Voice input werkt perfect in combinatie met annotaties — teken iets en leg het uit via spraak!

Tabs & Groepen

Tandem ondersteunt geavanceerd tab management met kleuren en groepen.

Tab Navigatie

Gebruik Cmd+1 tot Cmd+9 om snel tussen tabs te wisselen. Cmd+T opent een nieuwe tab, Cmd+W sluit de huidige tab.

Tab Indicatoren

Tabs tonen verschillende indicatoren:

  • 👤 — Tab beheerd door jou (Robin)
  • 🧀 — Tab beheerd door Kees (AI)
  • 🎵 — Tab speelt audio af
  • 🔴 — Audio wordt opgenomen

Geheugen & Data

Tandem onthoudt alles wat je doet om je later beter te kunnen helpen.

Site Memory

Voor elke website slaat Tandem op: titel, inhoud, formulieren, links, en veranderingen over tijd. Zoek door je browse geschiedenis via de Memory tab in het Kees panel.

Form Memory

Tandem onthoudt welke formulieren je invult en kan deze later auto-invullen. Gevoelige data zoals wachtwoorden worden versleuteld opgeslagen.

Behavioral Learning

Tandem leert je browse gedrag (timing, muis bewegingen, type patronen) om menselijker gedrag na te bootsen bij geautomatiseerde acties.

Stealth & Privacy

Tandem is ontworpen om volledig onzichtbaar te zijn voor websites. Het lijkt precies op een normale Chrome browser.

Anti-Detectie Features

  • Canvas fingerprint randomisatie
  • WebGL renderer/vendor spoofing
  • Audio context fingerprint spoofing
  • Timing protection
  • Font enumeration masking
  • Humanized delays en gedragssimulatie
🛡️ Privacy Alle data blijft lokaal op je computer. Tandem verstuurt NOOIT data naar externe servers.

Tips & Tricks

Workflow Tips

Screenshot + Voice

Maak een screenshot met annotaties en leg het uit via voice — perfect voor complexe vragen aan Kees.

PiP Monitoring

Gebruik Cmd+P voor Picture-in-Picture om Kees' activiteit te monitoren terwijl je ander werk doet.

Tab Groepen

Organiseer gerelateerde tabs in kleuren — blauw voor werk, groen voor research, etc.

Form Memory

Laat Tandem formulieren onthouden zodat Kees ze later automatisch kan invullen.

Geavanceerde Features

Tandem heeft veel geavanceerde features die niet zichtbaar zijn in de UI:

  • Headless Browsing: Kees kan onzichtbaar websites bezoeken
  • Scheduled Watches: Automatisch websites controleren op veranderingen
  • Network Inspector: API discovery en request monitoring
  • Extension Support: Chrome extensies laden
  • Audio Capture: Tab audio opnemen voor later
